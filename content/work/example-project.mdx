# Example Project

This is an example of how MDX content works in your Next.js application.

## Features

- **Rich content**: Mix markdown with React components
- **Frontmatter support**: Metadata at the top of files
- **Custom styling**: Tailwind CSS classes applied automatically

## Code Example

Here's a simple React component example:

```jsx
function MyComponent() {
  return <div>Hello from MDX!</div>
}
```

## Benefits

1. **Flexibility**: Write content in markdown with React components
2. **Performance**: Server-side rendering with Next.js
3. **Type Safety**: Full TypeScript support
4. **SEO**: Static generation for better search engine optimization
